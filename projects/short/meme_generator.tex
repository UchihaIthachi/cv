\nointent
An automated system for generating contextually relevant memes from any URL, leveraging LLMs and the Memegen.link API.
\begin{itemize}
    \item Orchestrated a multi-step meme generation workflow using LangGraph.
    \item Utilized Llama 3.3 (via Groq API) for content analysis and text creation.
\end{itemize}

\nointent
\textit{Keywords:} Streamlit, LangGraph, Llama 3.3, Groq
