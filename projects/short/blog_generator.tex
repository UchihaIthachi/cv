\subsection*{AI Blog Post Generator}
A multi-agent system that uses specialized AI agents to generate professional, well-structured, and well-researched blog posts.
\begin{itemize}
    \item Built with the Agno Framework for its beginner-friendly approach to multi-agent workflows.
    \item Designed for provider flexibility, with easy switching between OpenAI, Gemini, Claude, and Groq models.
\end{itemize}
\textit{Keywords:} Streamlit, Agno Framework, Pydantic
