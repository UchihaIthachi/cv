A serverless microservices platform built on AWS with event-driven architecture.
\begin{itemize}
    \item Architected using AWS Lambda, API Gateway, DynamoDB, and EventBridge, with SQS/SNS for async messaging.
    \item Implemented custom JWT Lambda Authorizers and used LocalStack/Docker for local development.
\end{itemize}

\nointent
\textit{Keywords:} Node.js, AWS Lambda, DynamoDB, SQS, SNS, SES, JWT, LocalStack, Docker
