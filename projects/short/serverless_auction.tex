\subsection*{Serverless Auction Platform \href{https://github.com/UchihaIthachi/serverless-auction-platform}{\faGithub}}
A capstone project built using AWS, following a microservices architecture with an event-driven, serverless (FaaS) design.
\begin{itemize}
    \item Architected an event-driven system with independently deployable microservices for auctions, authentication, and notifications.
    \item Leveraged AWS services including Lambda, API Gateway, DynamoDB, SQS, and SNS, deployed with the Serverless Framework.
\end{itemize}
\textit{Keywords:} Node.js, Serverless Framework, AWS Lambda, DynamoDB, SQS, SNS
