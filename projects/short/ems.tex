\subsection*{Employee Management System (EMS) \href{https://github.com/UchihaIthachi/EMS}{\faGithub}}
Designed and built a comprehensive, cloud-native microservices application for employee and department management, featuring a full-stack architecture with a React/TypeScript frontend and Spring Boot backend.
\begin{itemize}
    \item Architected a distributed system with Spring Cloud (Eureka, Config Server, Gateway), Feign Clients, and RabbitMQ.
    \item Established a full observability stack (Zipkin, ELK, Prometheus/Grafana) and a CI/CD pipeline with Jenkins and ArgoCD for Kubernetes deployment.
\end{itemize}
\textit{Keywords:} Java 17, Spring Boot, Spring Cloud, React, TypeScript, RabbitMQ, Docker, Kubernetes, Jenkins, ArgoCD
