\subsection*{Food Delivery Application \href{https://github.com/FoodDeliveryApp-ISA}{\faGithub}}

\nointent
Built a scalable, event-driven food delivery platform using a modern microservices architecture.
\begin{itemize}
    \item Developed the Restaurant Module with secure sign-up/login (Spring Security, JWT) and CRUD APIs with Redis caching.
    \item Enabled real-time order sync and notifications using Kafka and WebSockets.
\end{itemize}

\nointent
\textit{Keywords:} Java, Spring Boot, gRPC, Kafka, Redis, WebSocket, Docker
