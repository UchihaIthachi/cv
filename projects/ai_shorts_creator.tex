\subsection*{Automated AI Shorts Video Creator}

\nointent
An open-source project to automate the creation of short-form videos using a powerful combination of AI tools and workflow automation.

\begin{itemize}
    \item Orchestrated a video creation pipeline using \textbf{n8n} as the central workflow engine.
    \item Integrated various AI services for script generation, text-to-speech, and content preparation, including \textbf{Together AI}, \textbf{OpenRouter}, and \textbf{ElevenLabs}.
    \item Containerized the entire stack using \textbf{Docker Compose} for one-click local deployment.
    \item Implemented notifications via \textbf{Telegram} to monitor the automation workflow.
\end{itemize}

\nointent
\textit{Keywords:} n8n, Docker, AI, Workflow Automation, Video Generation
