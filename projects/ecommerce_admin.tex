Comprehensive admin interface for the Maison Vella e-commerce ecosystem, facilitating centralized management of products, orders, and content assets.

\begin{itemize}
    \item Engineered a responsive \textbf{Next.js 14} application using the \textbf{App Router}, \textbf{ShadCN UI}, and \textbf{Tailwind CSS}, ensuring a consistent and accessible design system.
    \item Implemented a robust backend architecture utilizing \textbf{Server Actions} for form handling and \textbf{tRPC} with \textbf{TanStack Query} for type-safe client-server communication.
    \item Orchestrated content management by integrating \textbf{Sanity} as a \textbf{Headless CMS}, consuming data via \textbf{GraphQL} for flexible and efficient retrieval.
    \item Managed transactional data using \textbf{Prisma ORM} with \textbf{Supabase (PostgreSQL)} in a \textbf{Serverless} environment.
    \item Architected the system using \textbf{CQRS}, the \textbf{Repository Pattern}, and a \textbf{Layered Architecture} within a feature-sliced structure to ensure scalability and maintainability.
    \item Secured administrative access and role-based operations using \textbf{Clerk Auth}.
    \item Optimized asset delivery and storage by implementing \textbf{Cloudflare R2} (S3-compatible) and a global \textbf{CDN} for high-performance media handling.
\end{itemize}

\nointent
\textit{Keywords:} Next.js 14 (App Router), Server Actions, TypeScript, tRPC, GraphQL, Prisma, Supabase, Sanity (Headless CMS), Serverless, Cloudflare R2 (CDN), CQRS
