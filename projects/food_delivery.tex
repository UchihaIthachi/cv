\subsection*{Food Delivery Application  \href{https://github.com/FoodDeliveryApp-ISA}{\faGithub}}
Scalable, \textbf{event-driven microservices} food delivery platform developed during internship at ISA, emphasizing real-time communication, high performance, and secure data handling.

\begin{itemize}
    \item Designed and implemented the \textbf{Restaurant Service} as part of a distributed microservices architecture.
    \item Developed and secured restaurant registration/login using \textbf{Spring Security} with \textbf{JWT}.
    \item Implemented CRUD menu management with \textbf{Redis} caching for improved performance.
    \item Enabled real-time order updates via \textbf{WebSocket} notifications.
    \item Integrated \textbf{gRPC} for synchronous and \textbf{Kafka} for asynchronous, \textbf{event-driven} service workflows.
    \item Orchestrated backend, frontend, and infrastructure (\textbf{Redis}, \textbf{Kafka}, \textbf{PostgreSQL}) using \textbf{Docker Compose}.
\end{itemize}

\textit{Tech:} Java 17, Spring Boot, gRPC, Kafka, Redis, PostgreSQL, WebSockets, Docker, Docker Compose, React, Ant Design
