Comprehensive admin interface for the Maison Vella e-commerce ecosystem, facilitating centralized management of products, orders, and content assets.

\begin{itemize}
    \item Engineered a responsive \textbf{Next.js 14} app with \textbf{Server Actions}, \textbf{tRPC}, and \textbf{ShadCN UI}, implementing \textbf{CQRS} and layered architecture.
    \item Integrated \textbf{Sanity} (Headless CMS) via GraphQL and managed data with \textbf{Prisma} and \textbf{Supabase} in a serverless environment.
    \item Secured access with \textbf{Clerk Auth} and optimized media delivery using \textbf{Cloudflare R2} and a global CDN.
\end{itemize}
