\subsection*{AI Blog Post Generator}

\nointent
A multi-agent system that uses specialized AI agents to generate professional, well-structured, and well-researched blog posts.

\begin{itemize}
    \item Built with the \textbf{Agno Framework} for its beginner-friendly approach to multi-agent workflows.
    \item Orchestrated research, content extraction, and writing using specialized agents for each task.
    \item Designed for provider flexibility, with easy switching between \textbf{OpenAI, Gemini, Claude, and Groq} models.
\end{itemize}

\nointent
\textit{Keywords:} Streamlit, Agno Framework, Pydantic
