\subsection*{Content-Aware Meme Generator (2025 | Individual) \hfill \href{https://github.com/UchihaIthachi/Meme_Generator}{\faGithub}}

\nointent
An automated system for generating contextually relevant memes from any URL, leveraging LLMs and the Memegen.link API.

\begin{itemize}
    \item Orchestrated a multi-step meme generation workflow using \textbf{LangGraph}.
    \item Utilized \textbf{Llama 3.3} (via Groq API) for content analysis, concept generation, and text creation.
\end{itemize}

\nointent
\textit{Keywords:} Streamlit, LangGraph, Llama 3.3, Groq, Pydantic, Beautiful Soup
