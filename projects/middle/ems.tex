\subsection*{Employee Management System (EMS) \href{https://github.com/UchihaIthachi/EMS}{\faGithub}}
Designed and built a comprehensive, cloud-native microservices application for employee and department management, featuring a full-stack architecture with a React/TypeScript frontend and Spring Boot backend.

\begin{itemize}
    \item Architected a distributed system with independent microservices for employees, departments, and configuration, using \textbf{Spring Cloud} for service discovery (\textbf{Eureka}), centralized configuration, and an API gateway.
    \item Established a robust observability stack, integrating \textbf{Zipkin/Sleuth} for distributed tracing, the \textbf{ELK Stack} for centralized logging, and \textbf{Prometheus/Grafana} for real-time metrics and monitoring.
    \item Designed and managed a CI/CD pipeline using \textbf{Jenkins}, with \textbf{ArgoCD} for GitOps-based deployments to a production-like \textbf{Kubernetes (OKE)} cluster.
\end{itemize}

\textit{Keywords:} Java 17, Spring Boot, Spring Cloud, React, TypeScript, Docker, Kubernetes, Jenkins, ArgoCD, Prometheus, Grafana, ELK Stack
