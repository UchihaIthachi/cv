\ProjectTitle{Employee Management System (EMS) (2025 | Individual) \hfill \href{https://github.com/UchihaIthachi/EMS}{\faGithub}}


\nointent
Designed and built a comprehensive, cloud-native microservices application for employee and department management, featuring a full-stack architecture with a React/TypeScript frontend and Spring Boot backend.

\begin{itemize}
    \item Architected a distributed system using \textbf{Spring Cloud} for service discovery, centralized configuration, and an API gateway, with \textbf{Resilience4j} for robust fault tolerance.
    \item Enabled dynamic configuration management using \textbf{Spring Cloud Bus} with \textbf{RabbitMQ} and established a robust observability stack (Zipkin, ELK, Prometheus/Grafana).
    \item Designed and managed a CI/CD pipeline using \textbf{GitHub Actions}, with \textbf{ArgoCD} for GitOps-based deployments to the Kubernetes cluster.
\end{itemize}

\nointent
\textit{Keywords:} Java 17, Spring Boot, Spring Cloud, React, TypeScript, Resilience4j, RabbitMQ, Docker, Kubernetes, GitHub Actions, ArgoCD, Prometheus, Grafana
