\nointent 
Built a full-stack decentralized application (DApp) on Polygon Amoy, enabling end-to-end shipment tracking and secure escrow payments via smart contracts. 
 
\begin{itemize} 
    \item Developed a production-grade \textbf{Solidity} smart contract (\texttt{Tracking.sol}) with a strict 3-state lifecycle and guarded state transitions. 
    \item Implemented a \textbf{trustless escrow mechanism} validating \texttt{msg.value} and releasing funds only upon verified delivery. 
    \item Built a \textbf{Next.js} frontend with \textbf{Ethers.js} and \textbf{Web3Modal} for wallet connection and real-time state management. 
    \item Designed a manual \textbf{CI/CD pipeline} using \textbf{GitHub Actions} for contract and frontend deployment to \textbf{Vercel}. 
    \item Utilized \textbf{Hardhat} for local blockchain simulation, testing, and contract verification. 
\end{itemize} 
 
\nointent 
\textit{Keywords:} Solidity, Hardhat, Next.js, Ethers.js, Web3Modal, Polygon Amoy, Smart Contracts, Escrow, CI/CD
