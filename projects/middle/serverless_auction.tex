\subsection*{Serverless Auction Platform \href{https://github.com/UchihaIthachi/serverless-auction-platform}{\faGithub}}
A capstone project built using AWS, following a microservices architecture with an event-driven, serverless (FaaS) design.

\begin{itemize}
    \item Architected an event-driven system with independently deployable microservices for auctions, authentication, and notifications.
    \item Leveraged a full suite of AWS services, including \textbf{Lambda}, \textbf{API Gateway}, \textbf{DynamoDB}, \textbf{SQS}, \textbf{SNS}, and \textbf{SES}.
    \item Utilized the \textbf{Serverless Framework} for streamlined deployment and infrastructure management on AWS.
\end{itemize}

\textit{Keywords:} Node.js, Serverless Framework, AWS Lambda, API Gateway, DynamoDB, SQS, SNS, SES, JWT
