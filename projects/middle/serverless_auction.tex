\subsection*{Serverless Auction Platform \href{https://github.com/UchihaIthachi/serverless-auction-platform}{\faGithub}}

\nointent
A capstone project built using AWS, following a microservices architecture with an event-driven, serverless (FaaS) design.

\begin{itemize}
    \item Architected an event-driven system using \textbf{AWS Lambda}, \textbf{API Gateway}, and \textbf{EventBridge}, with \textbf{SQS} and \textbf{SNS} for asynchronous communication.
    \item Designed a high-performance \textbf{DynamoDB} data model with GSIs and implemented a custom \textbf{Lambda Authorizer} for JWT-based security.
    \item Utilized \textbf{LocalStack} and \textbf{Docker} for local cloud emulation and integrated \textbf{Middy} middleware for robust error handling.
\end{itemize}

\nointent
\textit{Keywords:} Node.js, Serverless Framework, AWS Lambda, DynamoDB, SQS, SNS, SES, JWT, Middy, LocalStack, Docker
