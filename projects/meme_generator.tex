An AI-powered tool that generates memes based on image context.

\begin{itemize}
    \item Utilized \textbf{OpenAI GPT-4 Vision} to analyze images and generate humorous captions.
    \item Integrated \textbf{Pillow (PIL)} for image processing and text overlay.
    \item Created a simple web interface using \textbf{Streamlit} for user uploads.
    \item Implemented error handling and rate limiting for API requests.
\end{itemize}

\nointent
\textit{Keywords:} Python, OpenAI Vision API, Pillow, Streamlit, AI, Image Processing
