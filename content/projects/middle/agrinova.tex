\ProjectTitle{Agrinova Inventory Management System (2024 | Freelance)}

\noindent
Designed and delivered a production-grade inventory and sales management system for a real-world client, supporting multi-branch stock operations, invoicing, receipts, and commission tracking.

\begin{itemize}
    \item Engineered transaction-safe stock movement logic and replaced application-layer aggregation with \textbf{PostgreSQL} aggregations, reducing execution time from \textbf{$\sim$384 ms to $\sim$0.02 ms}.
    \item Designed a multi-layer caching strategy with \textbf{Next.js} and \textbf{React Query}, reducing repeated dashboard latency from \textbf{$\sim$703 ms to $\sim$0.05 ms}.
    \item Built a clean, dynamic admin-focused UI and validated system reliability through end-to-end testing with \textbf{Playwright}.
\end{itemize}

\noindent
\textit{Keywords:} Next.js, PostgreSQL, Prisma, React Query, Tailwind CSS, Playwright
